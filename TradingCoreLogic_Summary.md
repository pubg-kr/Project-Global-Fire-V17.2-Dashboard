## 4. 마스터 알고리즘 (Priority Action)
**"True가 나오면 실행 후 즉시 종료 (Else If 구조)"**

1.  **[Priority 1] 생존 헌법:** 수익률 **+1.5% 미만**이면 어떤 신호가 와도 **절대 매도 금지**.
2.  **[Priority 2] 겨울 프로토콜:** **일봉 200선** 붕괴 시 **현금 50%** 확보 (부족 시에만 매도).
3.  **[Priority 2.5] 봄의 귀환:** 겨울 → 봄 전환 (200일선 회복) 시, **4주 분할 매수**로 안전하게 복귀.
    *   *Abort:* 매수 도중 다시 200일선 깨지면 즉시 중단 & 현금 50% 재확보.
4.  **[Priority 3] 스나이퍼 (Sniper):** 역피라미드 분할 매수.
    *   **-15% (10%)** / **-25% (20%)** / **-35% (30%)** / **-45% (40%)**
    *   *겨울엔 -15% 구간 패스하고 -25%부터 진입.*
4.  **[Priority 4] 광기 차단:** (봄) RSI 80 이상 → **부분 익절** (Target + 10%).
5.  **[Priority 5] 듀얼 리밸런싱:** TQQQ vs USD **50:50** 유지 (괴리율 10%p).
